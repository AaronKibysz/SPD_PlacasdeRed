TARJETA DE INTERFAZ DE RED

Una tarjeta de interfaz de red (NIC) es un componente de hardware, normalmente una placa de circuito o chip, que se instala en un ordenador para que pueda conectarse a una red. Las NIC modernas proporcionan funcionalidades a los ordenadores, como soporte para interrupción de E/S, interfaces de acceso directo a memoria (DMA), transmisión de datos, ingeniería de tráfico de red y particionamiento. Cada tarjeta representa un dispositivo y puede preparar, transmitir y controlar el flujo de datos en la red.
